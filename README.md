### Gulp

`npm install --only=dev`
`gulp --sass`
`gulp --javascript`
`gulp`
`gulp --style=`

Style configuration
```javascript
AssetSASS:{
  debugInfo: true,
  lineNumbers: false,
  errLogToConsole: true,
  sourceComments: "normal",//normal, map
  outputStyle: "compact"//nested, compact, compressed, expanded
},
```

Script configuration
```javascript
AssetJS:{
  mangle:false,
  output:{
    beautify: true,
    comments:"license"
  },
  compress:true,
  outSourceMap: true,
  preserveComments:"license"
},
```
